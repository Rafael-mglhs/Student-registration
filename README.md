# Student-registration
A simple console application developed to practice fundamental c# concepts

## Concepts Practiced
- Classes
- Lists
- Loops
- Conditional Statements
- User Input Handling

## Features

- Register students
- List registred students
- Search for a student
- Remove a student
- Exit program

## Technologies

- C#
- .NET 10
- Console Application

## Running The Project

Clone the repository
 git clone https://github.com/Rafael-mglhs/Student-registration.git

Navigate to project folder
 cd CadastroAlunos

Run
 dotnet run
